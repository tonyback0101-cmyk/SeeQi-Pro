# 替换 icon-192.png 图标

## 快速替换步骤

1. **准备图标文件**
   - 确保图标文件是 192x192 像素的 PNG 格式
   - 文件名必须是 `icon-192.png`

2. **替换文件**
   - 将新的图标文件复制到 `public/icons/` 目录
   - 替换现有的 `icon-192.png` 文件

3. **验证**
   - 访问 `http://localhost:3000/icons/icon-192.png` 确认文件可以正常加载
   - 检查文件大小（应该比 67 字节大得多）

## 文件位置

```
public/icons/icon-192.png
```

## 图标要求

- **尺寸**: 精确的 192x192 像素
- **格式**: PNG（支持透明背景）
- **文件大小**: 通常应该在 5-50 KB 之间（取决于压缩率）

## 注意事项

- 替换后需要重启开发服务器（如果正在运行）
- 清除浏览器缓存以确保显示新图标
- 测试 PWA 安装功能确认图标正确显示
