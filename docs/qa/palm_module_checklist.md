# Palm Module QA Checklist

## 功能用例

- [ ] 上传单张掌纹（摄像头拍摄）
- [ ] 上传单张掌纹（本地图片）
- [ ] 触发文件大小/格式错误
- [ ] 删除掌纹记录
- [ ] 标注特征新增/拖拽/删除/保存
- [ ] 标注缩放及小屏交互

## 批量导入

- [ ] 多文件批量上传成功
- [ ] 批量文件中单个标注修改
- [ ] 批量导入中途删除某项
- [ ] 批量导入失败/提示总结

## 离线场景

- [ ] 无网络上传入队成功
- [ ] 恢复网络自动同步
- [ ] 手动触发同步按钮
- [ ] 离线批量导入入队
- [ ] `/[locale]/analyze` 离线提交后返回缓存提示
- [ ] `/[locale]/analysis-result/[id]` 离线刷新展示最近缓存报告

## 异常与恢复

- [ ] Supabase 无权限或认证失效
- [ ] API 超时/网络慢导致队列提示
- [ ] 重新登录后队列保持
- [ ] PWA 安装后离线访问 `/privacy` 和最近报告成功

## 验收记录

- [ ] 版本与日期
- [ ] 执行人
- [ ] 结果汇总
