# Palm Module User Guide

## 1. 上传掌纹
1. 登录后进入“掌纹采集”页面。
2. 选择手型、掌纹区域与采集方式。
3. 点击“上传掌纹”或“批量上传”。
4. 若网络离线，系统会提示掌纹被暂存，重新联网后会自动同步。

## 2. 批量上传
1. 点击“批量上传”卡片中的“选择文件”。
2. 设置默认信息（手型/区域/方式/质量）。
3. 如需单独修改，可在列表中逐项调整。
4. 点击“批量上传”后，系统会逐张提交。

## 3. 离线同步
- 离线上传会显示在“离线上传队列”中。
- 网络恢复时自动同步，也可点击“立即同步”。
- 如连续失败，请联系支持或手动重试。

## 4. 特征标注
1. 在列表中选择掌纹，点击“标注特征”。
2. 点击图片新增标注点，可拖动或编辑描述。
3. 使用右下角缩略图或滑杆调整缩放。
4. 保存后可在报告中查看特征信息。

## 5. 常见问题
- 无法访问摄像头：浏览器或系统权限需允许。
- 上传失败：检查网络状态或重试，持续失败时联系支持。
- 质量评分：范围 1-5，可留空表示未评估。
